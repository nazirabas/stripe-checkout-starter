# Security Policy

## Reporting a vulnerability

If you find a security issue, please **do not open a public GitHub issue.**

Email **nazirabbas1995@gmail.com** with:

- A description of the issue
- Steps to reproduce
- The version or commit SHA where you saw it
- Your suggested fix, if you have one

I aim to respond within 72 hours.

## Supported versions

This is a starter repo, not a maintained library. The latest commit on `main` is what's supported. Older versions are not backported.

If you fork and ship to production, treat this as a starting point — you own the security posture of your fork.
